# Фоторамка для показу фотоальбому із google photo
- Необмежена кількість фото
- 5 ефектів зміни фото
- вибір тривалості показу 1 слайду
- автоматичний підбір розміру фото під дисплей
- присутній random режим
Перед прошиванням змінити ім'я та пароль на wi-fi.
Після прощивання зайти на сторінку по адресу на дисплеї, вставити посилання га альбом.
